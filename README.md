# Cross-Community Tech Talent Synthesis Report

An interactive web application showcasing insights from six rural communities building tech talent pipelines through employer roundtables and educator discussions.

## Communities Featured

- **Ada, Oklahoma** - Pontotoc Technology Center & East Central University partnerships
- **Chambers County, Alabama** - Manufacturing tech & digital literacy focus
- **Cochise County, Arizona** - Cybersecurity pathways with Fort Huachuca
- **Selma, Alabama** - Remote work readiness & Best Buy Teen Tech Center
- **Taos, New Mexico** - Cultural partnerships & wraparound supports
- **Wilson, North Carolina** - Gig East Exchange & Workforce Alliance

## Technology Stack

- **React 18** with TypeScript
- **Vite** for fast development
- **Tailwind CSS** with custom design system
- **shadcn/ui** component library
- **Recharts** for data visualization
- **React Router** for navigation

## Design System

All colors use HSL semantic tokens defined in `src/index.css`:
- Community-specific colors (Ada, Chambers, Cochise, Selma, Taos, Wilson)
- Status colors (primary, success, warning, accent)
- Glass-card effects for modern UI
- Responsive grid layouts
- Smooth animations and transitions
